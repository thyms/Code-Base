Ubuntu 11.04 Installation After-works
-------------------------------------
--> Choose "Bridged Adaptor" from Settings->Network

--> VirtualBox Guest Utils
$sudo apt-get update
$sudo apt-get install virtualbox-ose-guest-utils


--> Windows 95-ish theme problem
-> Solution 1, requires every time rebooted.
$killall -9 gnome-settings-daemon && gnome-settings-daemon // For windows 95-ish theme fix
-> Solution 2, permanent
$gksudo gedit /etc/xdg/autostart/gnome-settings-daemon.desktop
- Change the line:
Exec=/usr/lib/gnome-settings-daemon/gnome-settings-daemon
- with this:
Exec=bash -c "sleep 15; /usr/lib/gnome-settings-daemon/gnome-settings-daemon"


--> Shared Folder, Mounting
-> You have to add "shared" folder from VirtualBox->Devices->Shared Folder
$sudo mkdir ~/Shared
$sudo mount.vboxsf shared ~/Shared
-> To make it mounted after every reboot
$sudo gedit /etc/rc.local
- Add this line:
mount.vboxsf shared ~/Shared


--> Resolution problem, fix(1440x900)
- Change xorg.conf file and reboot.
File->/etc/X11/xorg.conf

Installing Java JDK on Ubuntu 11.04
-----------------------------------
$sudo add-apt-repository ppa:ferramroberto/java
$sudo apt-get update
$sudo apt-get install sun-java6-jdk
- for JRE
$sudo apt-get install sun-java6-jre sun-java6-plugin
